# Project Title

NFT Marketplace – A Soroban Smart Contract dApp for Buying and Selling NFT Art Pieces on Stellar

## Project Vision

This project demonstrates a decentralized NFT marketplace built on the Stellar blockchain using Soroban smart contracts. It provides a platform where artists can list their NFT art pieces for sale and buyers can purchase them directly using the marketplace contract.

---

## Description

A Soroban smart contract dApp that enables users to **buy and sell NFT art pieces** on Stellar Testnet. The marketplace allows NFT owners to list their tokens with a price, buyers can purchase listed NFTs, and anyone can query the listing status of any token.

---

## Features

### 1. List NFTs for Sale
- NFT owners can list their tokens with a specified price
- Listings are stored permanently on-chain
- Owner authentication required for listing

### 2. Buy NFTs
- Buyers can purchase listed NFTs by paying the asking price
- Listings are removed once sold
- Buyer authentication required for purchases

### 3. Query Listings
- Anyone can check if an NFT is listed for sale
- Returns the owner and price if listed
- Transparent on-chain data

### 4. Cancel Listings
- Owners can delist their NFTs at any time
- Owner authentication required for cancellation
